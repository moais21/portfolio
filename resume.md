Проект демонстрирует навыки работы с SQL на примере базы данных системы учёта фотостудии.

**Структура БД:**
- `users` — пользователи
- `halls` — залы для аренды
- `bookings` — бронирования
- `tariffs` — тарифы
- `payments` — платежи
- `reviews` — отзывы

**Файлы проекта:**
- `schema.sql` — DDL-скрипт создания таблиц
- `data.sql` — DML-скрипт заполнения тестовыми данными
- `queries.sql` — 15 SQL-запросов разной сложности

**Что демонстрирует проект:**
- Проектирование реляционной БД
- Написание запросов с JOIN, GROUP BY, подзапросами
- Оконные функции
- Аналитические запросы для бизнес-отчётности
